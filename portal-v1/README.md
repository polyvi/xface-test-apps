## 验证方法：
  1. portal能够正确加载并显示__推荐应用列表__
  2. 点击__推荐列表__中的应用图标能够下载并安装应用；再次点击应用图标，能够启动应用
  3. 进入__本地应用__能够查看已安装应用图标
  4. 进入__本地应用__点击图标能够启动应用，长按图标能够卸载应用
  5. 进入__设置__能够卸载所有本地应用
